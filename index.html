Dispositivo

* SHIFT + Arrastar de um nó a outro para criar cabos.

* Clique num roteador para desligá-lo.

* Botão Direito para Tabela RIPv2 ou Excluir.

* Delete exclui o nó selecionado.

Esta rede utiliza o protocolo OSPF. O caminho é definido pela Velocidade (Largura de Banda) do cabo (Fibra é mais rápido que Rádio).

Resiliência: Temos rotas redundantes. Clique num link para "quebrá-lo" e simule novamente para ver o desvio!
